# Monetization Model

## Access Tiers

- Basic: Free listing, capped usage
- Pro: Negotiation + override routing
- Legacy: Council protection + emotional flag coverage

## Fees

| Flow Type              | Fee (%) |
| ---------------------- | ------- |
| Basic Listing          | 0       |
| Negotiation + Override | 5       |
| Financial Execution    | 3       |
| Relationship Rituals   | 8       |
| Council-Approved Trade | 6       |

## Notes

- Fees are override-safe and emotionally scoped
- Council reviews fee structure monthly
- Founder blessing required for changes
